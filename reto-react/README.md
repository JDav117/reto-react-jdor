# Simple Coffee Listing

Aplicación React que consume una API y muestra un catálogo dinámico de cafés con filtrado por disponibilidad.
Reto: [Simple Coffee Listing — devChallenges](https://devchallenges.io/challenge/simple-coffee-listing)

## Integrantes

- Jhoan David Ortega Ramos

## Instrucciones para ejecutar

- npm install (para instalar dependencias).
- npm run dev (Correr en terminal y posterior visualizacion en navegador).

Abre http://localhost:5173 en el navegador.
